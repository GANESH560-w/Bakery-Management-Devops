### Bakery Project: Complete AWS Deployment Guide Step 1: AWS Infrastructure (Manual Setup)
1. Create AWS RDS (MariaDB) :
   - Engine: MariaDB (Free Tier).
   - Set your Username and Password .
   - Enable Public Access (Yes).
   - Copy the Endpoint once created.
2. Create AWS EC2 (Ubuntu) :
   - Instance Type: t2.micro .
   - Security Group Rules (Inbound) :
     - Port 80 (HTTP)
     - Port 8080 (Custom TCP)
     - Port 22 (SSH)
3. Link EC2 to RDS : In the RDS Security Group, allow Port 3306 for the EC2 Security Group. Step 2: Database Table Setup
Connect to your EC2 via SSH and run these commands to prepare the database:

```
# 1. Install MySQL Client
sudo apt update && sudo apt install 
mysql-client -y

# 2. Log into RDS (Replace 
placeholders)
mysql -h <YOUR_RDS_ENDPOINT> -P 
3306 -u <RDS_USER> -p

# 3. Run these SQL commands:
CREATE DATABASE bakery_db;
USE bakery_db;
CREATE TABLE contact_messages (
    id INT AUTO_INCREMENT PRIMARY 
    KEY,
    payload TEXT,
    created_at TIMESTAMP DEFAULT 
    CURRENT_TIMESTAMP
);
EXIT;
``` Step 3: EC2 Server Initialization
Install the required software and get the project code:

```
# 1. Install Java, Nginx, and Git
sudo apt install openjdk-17-jdk 
nginx git -y

# 2. Clone the project
git clone <YOUR_REPO_URL>
cd BakeryProject
``` Step 4: Frontend Deployment (Nginx)
Serve the website using Nginx:

```
# 1. Copy frontend files to web root
sudo cp -r baker-1.0.0/* /var/www/
html/

# 2. Set permissions
sudo chown -R www-data:www-data /
var/www/html
sudo chmod -R 755 /var/www/html

# 3. Restart Nginx
sudo systemctl restart nginx
``` Step 5: Backend Deployment (Temporary Method)
Important: You must run these export commands every time you open a new terminal session.

```
# 1. Set Temporary Environment 
Variables (REPLACE with your 
details)
export DB_URL="jdbc:mysql://
<YOUR_RDS_ENDPOINT>:3306/bakery_db"
export DB_USER="admin"
export DB_PASS="your_password"

# 2. Navigate to backend and compile
cd ~/BakeryProject/baker-1.0.0/
backend-java
javac -cp "lib/mysql-connector-j-8.
3.0.jar" -d out src/*.java

# 3. Start the server
java -cp "out:lib/
mysql-connector-j-8.3.0.jar" 
BackendServer
``` Step 6: Verification & Access
- Frontend URL : http://<YOUR_EC2_PUBLIC_IP>/
- Backend API : http://<YOUR_EC2_PUBLIC_IP>:8080/
To verify data storage, run this on EC2:

```
mysql -h <RDS_ENDPOINT> -u 
<RDS_USER> -p -e "SELECT * FROM 
bakery_db.contact_messages;"
```
